Promptfy API Documentation

Welcome to the Promptfy API. This API provides authentication and AI-powered prompt generation using Gemini AI.

System Overview

Promptfy is a powerful prompt engineering platform that helps users create well-structured, effective prompts for AI models. The system consists of:

  • Authentication System - Secure user registration and login
  • Prompt Templates - Pre-defined structures for different types of prompts
  • Prompt Generation - AI-powered generation using Gemini
  • Prompt Enhancement - AI-based improvement of existing prompts
  • Prompt Analysis - Evaluation of prompt effectiveness

Authentication

POST /api/v1/auth/register

Register a new user

{
  "name": "John Doe",
  "email": "john@example.com",
  "password": "password123"
}
POST /api/v1/auth/verify-email

Verify user email

{
  "email": "john@example.com",
  "verificationToken": "token-received-in-email"
}
POST /api/v1/auth/login

Login with email and password

{
  "email": "john@example.com",
  "password": "password123"
}
POST /api/v1/auth/forgot-password

Request password reset

{
  "email": "john@example.com"
}
POST /api/v1/auth/reset-password

Reset password with token

{
  "email": "john@example.com",
  "token": "reset-token-from-email",
  "password": "newpassword123"
}
GET /api/v1/auth/me

Get current user details (requires authentication)

Header: Authorization: Bearer your-token-here

Prompt Generation

GET /api/v1/prompts/templates

Get all prompt templates

Header: Authorization: Bearer your-token-here

GET /api/v1/prompts/templates/:id

Get prompt template by ID

Header: Authorization: Bearer your-token-here

POST /api/v1/prompts/templates

Create a new prompt template

Header: Authorization: Bearer your-token-here

{
  "name": "Template Name",
  "description": "Template description",
  "template": "Template content with structure",
  "category": "Category"
}
POST /api/v1/prompts/generate

Generate a prompt based on template and user instructions

Header: Authorization: Bearer your-token-here

{
  "title": "My Prompt",
  "userInstructions": "User instructions for prompt",
  "templateId": "template-id",
  "category": "Category",
  "tags": ["tag1", "tag2"]
}
GET /api/v1/prompts/user

Get all prompts created by the user

Header: Authorization: Bearer your-token-here

POST /api/v1/prompts/:id/enhance

Enhance an existing prompt

Header: Authorization: Bearer your-token-here

{
  "enhancementRequest": "Make it more detailed and add examples"
}
GET /api/v1/prompts/:id/analyze

Analyze a prompt for effectiveness

Header: Authorization: Bearer your-token-here
